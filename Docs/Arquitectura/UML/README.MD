# UML — DriveControl (Docs/Arquitectura/UML)

Esta carpeta contiene los **diagramas UML** del proyecto **DriveControl**, usados para documentar la arquitectura y el comportamiento del sistema (por ejemplo, el flujo de **registro de vehículos**).  
Los diagramas sirven como evidencia de análisis/diseño y apoyo para el desarrollo, revisión y entregables del curso.

## Contenido

Aquí se almacenan principalmente archivos en formato **PNG** (exportados) y, cuando aplique, sus fuentes editables (ej. `.puml`, `.drawio`, `.mmd`).
