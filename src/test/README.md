# Directorio de Pruebas

El directorio `tests` contiene los casos de prueba y validaciones funcionales del sistema Drive Control.

Su objetivo es garantizar el correcto funcionamiento del Dashboard de Cumplimiento, la gestión de flotas, la simulación de alertas y la validación de datos asociados a los documentos legales de los vehículos.

Las pruebas aseguran que la plataforma cumpla con los criterios de calidad definidos por SYNTIX TECH, minimizando riesgos operativos y asegurando la confiabilidad del sistema.
