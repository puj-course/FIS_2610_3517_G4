## Normalizacion de autores

Antes de presentar porcentajes, normalizar alias y correos duplicados. Un mismo integrante puede aparecer con variaciones de nombre, correo institucional, correo personal o usuario GitHub.

Comandos utiles:

```bash
git shortlog -sne --all
git log --all --format="%an <%ae>" | sort | uniq
```

En PowerShell:

```powershell
git shortlog -sne --all
git log --all --format="%an <%ae>" | Sort-Object | Get-Unique
```
