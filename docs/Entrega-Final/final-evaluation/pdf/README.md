# PDFs de evaluacion final

Esta carpeta es el destino de `scripts/generate_final_pdfs.sh`.

```bash
scripts/generate_final_pdfs.sh
```

Si el entorno no tiene `pandoc` o `markdown-pdf`, instala una de esas herramientas y vuelve a ejecutar el script. Los Markdown fuente completos estan en `docs/final-evaluation/`.
